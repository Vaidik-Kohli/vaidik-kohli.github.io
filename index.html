About

tetris_clone.py

Tetris Clone

Python · Pygame

Classic Tetris rebuilt from scratch in Python — seven tetrominoes, wall kicks, line clears, level progression, and a scoring system that gets mean fast.

The rotation logic was the interesting bit. Standard matrix rotation looks right on paper until pieces clip walls or floor. Implementing the SRS (Super Rotation System) wall kick table taught me more about state machines than three months of tutorials did.

Shakespeare Meme App

React · Anthropic API

Takes any modern text and returns an unhinged, over-the-top Shakespearean translation via the Anthropic API — built as a React app, lives in the browser.

The interesting part was prompt engineering: getting the API to commit fully to the bit without hedging or breaking character. There's a real gap between "sounds vaguely old" and "sounds genuinely unhinged-Elizabethan," and closing it took about fifteen iterations on the system prompt.

Reactor WIP

GDScript · Godot 4

A Godot 4 strategy/puzzle game where you manage a nuclear reactor under increasing pressure — balance heat, power output, and coolant flow before the timer hits zero.

First real project in GDScript, which means I've spent a lot of time learning Godot's scene system and signal architecture. The reactor simulation loop taught me how to model interdependent systems without turning update() into a mess.
